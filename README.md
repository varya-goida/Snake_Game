# Игра Змейка на Python

Классическая игра "Змейка" с графическим интерфейсом на tkinter.
## Архитектура проекта
snake-game/
├── main.py
├── constants.py
├── game_engine.py
├── ui_manager.py
├── dialogs.py
├── README.md
├── Dockerfile
└── .gitignore
## Особенности
- Выбор уровня сложности (легкий, средний, тяжелый)
- Ввод имени игрока
- Подсчет очков
- Управление с клавиатуры
- Русский интерфейс

## Установка и запуск

### Требования
- Python 3.6 или выше
- Tkinter (обычно входит в стандартную поставку Python)

## Запуск через Docker

### Сборка образа:
```bash
docker build -t snake-game .
